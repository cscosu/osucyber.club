# OSU Cyber Site
---
## Quickstart
1. Clone repo
2. `cd` into repo
3. `npm install`
4. `npm run build`
5. `npm start`
6. Configure nginx or whatever else you want to use to direct to localhost:8080
## Core tech stack

- React
- Semantic UI React

### Other

- React router
- React parallax
- React particles

#### To run

- `npm start`

#### To deploy to prod

- `npm run deploy`
